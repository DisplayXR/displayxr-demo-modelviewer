3D Model Viewer

Real-time glTF 2.0 PBR model viewer for glasses-free 3D displays, built on the DisplayXR OpenXR runtime via Vulkan. Loads .glb and .gltf models and renders with asymmetric per-eye Kooima projection.

Once installed, the app lands at /Applications/3D Model Viewer.app. A bundled sample.glb auto-loads at startup; press L or drag-and-drop another model to load it.

Prerequisite: the DisplayXR runtime must be installed first. Download DisplayXR-Installer-*.pkg from the displayxr-runtime releases page. The demo bundles its own copy of MoltenVK and the OpenXR loader, but it discovers the actual runtime through the system's OpenXR configuration registered by the runtime installer.
